# 🌧️ RainBeat

**RainBeat** is a modern, high-performance multimedia player application for Android, built entirely with Kotlin and **Jetpack Compose**. Designed for a seamless user experience, it acts as a unified hub for both local audio and video playback.

## ✨ Key Features

* **Unified Media Library:** Distinct screens for organizing and viewing your Music and Video libraries.
* **Folder Explorer:** Navigate your local device storage directory-by-directory to find specific media files with ease.
* **Continuous Playback:** Features a persistent `MiniPlayerBar` and a background `PlaybackService` to keep music playing even when navigating through different parts of the app or when the app is in the background.
* **High Performance:** Optimized data loading using background threads and carefully managed Jetpack Compose state to eliminate UI stuttering and reduce memory allocations during search and navigation.
* **Modern UI/UX:** A clean, responsive, and dynamic user interface following modern Android design guidelines.

## 🛠️ Technologies Used
* **Kotlin**
* **Jetpack Compose**
* **Coroutines & Flows**
* **Android Media3 / ExoPlayer**
